## Fun Commands
Fun commands are simple to use. Below is two examples.

### 8ball
Ask the magic discordbot a question!
```bash
# -- format -- #
8ball <question>

# -- example -- #
!8ball will my girlfriend ever get back with me?
```

If successful, it will shoot out a randomized answer

### TicTacToe
Play a game of tic-tac-toe against a friend or a bot!

```bash
# -- format -- #
tictactoe <user>

!tictactoe @legend
```
